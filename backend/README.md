# Backend APP

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Run tests

```bash
# unit tests
$ npm run test

```

## Docker

```bash
# build and run
$ docker build -t backend . && docker run -p 3000:3000 backend
```
